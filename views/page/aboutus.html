แนะนำ

คณะผู้จัดทำจึงได้คิดจะทำ API เพื่อให้บริการนักพัฒนาแอพพลิเคชั่นได้นำไปพัฒนาแอพพลิเคชั่นโดยจะให้บริการเกี่ยวกับพจนานุกรมภาษาไทย ให้ความหมายของคำ คำเหมือน และหมวดหมู่ของคำ โดยนอกจากส่วน API ที่คณะผู้จัดทำจะทำแล้วนั้น ยังมีส่วน Crowdsourcing ที่จะช่วยให้พจนานุกรมไทยของเรานั้นได้รับการพัฒนาอย่างต่อเนื่อง และครอบคลุมกับคำศัพท์ทุกยุคทุกสมัย โดยคาดหวังว่าโครงการนี้จะสามารถช่วยตอบโจทย์กับนักพัฒนาแอพพลิเคชั่น หรือผู้อื่นที่ต้องการใช้งานภาษาไทย และช่วยในการอนุรักษย์ภาษาไทย สนับสนุนให้คนใช้ภาษาไทย เข้าใจภาษาไทยอย่างถูกต้อง
